Box Sizing: How It Affects Layout

The box-sizing property in CSS determines how the total width and height of an element are calculated.

Content
content-box (default): The width and height include only the content. Padding and border are added outside of these dimensions.
border-box: The width and height include the content, padding, and border. This is often easier to work with for layout design.

In this example, the red box has a height and width of 100px. When using content-box, the padding and border are added outside, making the total size larger. With border-box, the padding and border are included within the 100px, keeping the total size consistent.
