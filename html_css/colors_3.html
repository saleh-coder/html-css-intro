Hexadecimal Colors

Hexadecimal (or Hex) colors use a 6-character code to represent colors in the format #RRGGBB, where each pair of characters represents a color component:

  • RR: Red component (00 to FF, where FF is 255 in decimal).
  • GG: Green component (00 to FF).
  • BB: Blue component (00 to FF).

For example, the code #32CD32 corresponds to a lime green color:

  • 32: Red value in hexadecimal (50 in decimal).
  • CD: Green value in hexadecimal (205 in decimal).
  • 32: Blue value in hexadecimal (50 in decimal).
