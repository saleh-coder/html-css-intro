RGBA Color Model:

The RGBA color model defines colors based on four components:

  • Red: The amount of red in the color (0 to 255). In this example, the value is 60.
  • Green: The amount of green in the color (0 to 255). In this example, the value is 60.
  • Blue: The amount of blue in the color (0 to 255). In this example, the value is 120.
  • Alpha: The transparency level of the color (0 to 1). 0 is fully transparent, and 1 is fully opaque. In this example, the value is 0.6, making the text semi-transparent.
